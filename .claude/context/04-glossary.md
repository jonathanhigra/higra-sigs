# Glossário HIGRA — SIGS

Termos e abreviações usados em tabelas, colunas e UI.

## Qualidade e SGQ
| Termo       | Significado                                           |
|-------------|-------------------------------------------------------|
| RNC / RQ03  | Reclamação de Cliente / Não Conformidade              |
| RQ49        | Nota de Oportunidade (melhoria interna)               |
| RQ80        | Auditoria interna                                     |
| RQ94        | Análise de Mudança                                    |
| GAC         | Gerenciamento de Ações Corretivas (Plano de Ação)     |
| SST         | Segurança e Saúde no Trabalho                         |

## Fabricação
| Termo       | Significado                                           |
|-------------|-------------------------------------------------------|
| CKL         | Checklist de Fabricação                               |
| BOB         | Bobinagem (enrolamento do motor)                      |
| CNJ_MOT     | Conjunto Motor (montagem)                             |
| ENS_HID     | Ensaio Hidroenergético                                |
| EXP         | Expedição                                             |
| PIN         | Pintura                                               |
| QLD         | Qualidade                                             |
| MNT         | Manutenção                                            |
| EMB         | Embalagem                                             |

## Engenharia / Produto
| Termo       | Significado                                           |
|-------------|-------------------------------------------------------|
| MTR         | Motor                                                 |
| BMB         | Bomba                                                 |

## Pós-venda
| Termo       | Significado                                           |
|-------------|-------------------------------------------------------|
| LAU         | Laudo Técnico                                         |
| ATN         | Atendimento (Assistência Técnica)                     |
| CHM         | Chamado                                               |

## CRM / Comercial
| Termo       | Significado                                           |
|-------------|-------------------------------------------------------|
| FNL         | Funil (pipeline de vendas)                            |
| ETP         | Etapa                                                 |
| NEG         | Negócio (deal)                                        |
| LED         | Lead                                                  |
| ORG         | Organização (empresa cliente)                         |

| PRD         | Produto                                               |

## Sistema / Infraestrutura
| Termo       | Significado                                           |
|-------------|-------------------------------------------------------|
| STM         | Sistema (módulo de segurança/permissões)              |
| FOCCO3I     | ERP Focco (sistema legado de pedidos)                 |
| HGRHML      | Schema owner do APEX original                         |
| APEX        | Oracle Application Express (framework legado)         |

## Prefixos de tabela
| Prefixo  | Uso                                                      |
|----------|----------------------------------------------------------|
| `HGR_`   | HIGRA — tabelas específicas da empresa                   |
| `BEG_`   | Base — cadastros e estruturas gerais                     |
| `STH_`   | Legado (Sistema Higra) — geralmente compartilhado        |
| `CFG_`   | Configuração                                             |
| `_CAD_`  | (meio) Cadastro — ex.: `HGR_STM_CAD_TIPO_USU`            |
| `_REG_`  | (meio) Registro / relação N:M                            |
| `_VW_`   | (meio) View                                              |
